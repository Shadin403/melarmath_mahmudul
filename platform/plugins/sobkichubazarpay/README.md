# SobkichuBazarPay Payment Gateway Plugin

SobkichuBazarPay is a PHP e-commerce payment gateway plugin designed for Botble CMS that integrates with the SobkichuBazar payment gateway.

## Features

- Easy installation and configuration
- Secure payment processing
- Multiple currency support (BDT, USD, EUR, GBP)
- Real-time payment status updates
- Bengali language support
- Payment verification system

## Installation

1. Copy the plugin folder to your Botble CMS `platform/plugins/` directory
2. Activate the plugin in the admin panel
3. Configure SobkichuBazarPay in Payment Settings

## Configuration

Get the following credentials from your SobkichuBazar account:

- API Key
- Secret Key  
- Brand Key

Enter these credentials in the admin panel's Payment Settings.

## API Endpoints

- **Payment Creation**: `https://secure.sobkichubazar.com.bd/api/payment/create`
- **Payment Verification**: `https://secure.sobkichubazar.com.bd/api/payment/verify`
